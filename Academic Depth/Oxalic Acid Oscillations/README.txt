This is a short Description to guide you through the folder:

a. Electrochemical Modelling Part 1.pdf: Describes the problem statement and the Modelling
b. Electrochemical Modelling Part 2.pdf: Solves the model under the special condition of superfast diffusion
c. Case1.m: Solves the case 1 described in the pdf
d. Case1anime.m: Solves the Case1 and makes an animation for the solution
e. bifurcation.m: Draws the bifurcation diagram for case 1
f. Bifurcation Diagram.png: Bifurcation diagram of case 1
g. .avi files: Outputs for the Case1anime file. 
h. MEM PPT.pptx: A powerpoint presentation explaining the project and its scope.
